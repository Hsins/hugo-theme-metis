## Getting Started

```bash
# Create a New Site
$ hugo new site TIL

# Download Metis from GitHub and Add It to Themes Directory of Your Site
$ cd TIL
$ git init
$ git submodule add https://github.com/Hsins/hugo-theme-metis themes/metis
```
